Connotate - Master Project

Sat, Feb 13, 2021 One-minute read https://github.com/jamalxo/tos-platform

Application for lawyers to collaborate analyzing Terms of Service documents. Makes Google Docs unnecessary.

Editor

The heart piece of the application is the editor where several users can create sections of the document and afterwards annotate them. An annotation can be a label or a legal opinion. Lastly users can also discuss, like and comment on all annotation made in the document.

Login Page

The authentication is done via JWT. There’s not much to tell here besides the original design.

Home Page

The home page displays all previous visited documents, the status of your uploaded documents and also the task you have been assigned by other lawyers

Lists

Lastly there are also lists of all tasks and documents of the application. This way you can access old documents and review previous annotation made.
